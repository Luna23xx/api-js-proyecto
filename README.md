# Proyecto Web con Consumo de API

Este proyecto consiste en una aplicación web desarrollada con JavaScript que consume una API para mostrar información dinámica.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Consumo de API externa
- Visualización de datos dinámicos
- Interfaz interactiva
